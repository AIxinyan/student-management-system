### 快速部署（推荐）

#### 方式一：使用快速启动脚本
```powershell

Get-Service -Name MongoDB | Start-Service
```

```powershell

cd backend
npm install

# 3. 启动后端服务
npm start

# 4. 打开前端页面
# 双击 frontend/index.html
```

## 📡 API 接口文档

### 认证相关

#### 1. 用户注册
```
POST /api/auth/register
Content-Type: application/json

{
  "username": "吴军",
  "password": "123456"
}

Response:
{
  "success": true,
  "message": "注册成功，即将跳转到登录页面",
  "data": {
    "username": "吴军",
    "role": "user"
  }
}
```

#### 2. 用户登录
```
POST /api/auth/login
Content-Type: application/json

{
  "username": "root",
  "password": "000000",
  "loginType": "admin"  // 或 "user"
}

Response:
{
  "success": true,
  "message": "登录成功",
  "data": {
    "username": "root",
    "role": "admin"
  }
}
```

### 学生管理

#### 3. 获取学生列表
```
GET /api/students

Response:
{
  "success": true,
  "data": [...],
  "message": "获取学生列表成功"
}
```

#### 4. 创建学生
```
POST /api/students
Content-Type: application/json

{
  "name": "张三",
  "studentId": "2024021",
  "class": "一班",
  "score": 88
}
```

#### 5. 更新学生
```
PUT /api/students/:id
Content-Type: application/json

{
  "score": 95
}
```

#### 6. 删除学生
```
DELETE /api/students/:id
```

#### 7. 筛选学生
```
GET /api/students/filter/search?class=一班&minScore=80
```

#### 8. AI 分析报告
```
GET /api/students/analysis/report
```

## 📚 使用培训

### 管理员培训

#### 1. 登录系统
1. 打开系统入口
2. 选择"管理员登录"
3. 输入 root/000000

#### 2. 管理学生

#### 3. 查看分析
1. 点击"AI 分析报告"
2. 查看各项统计数据
3. 阅读 AI 建议

#### 1. 注册账号
1. 点击"立即注册"
2. 输入学生姓名（必须是系统中的）
3. 设置密码（至少6位）
4. 确认密码
5. 注册成功后跳转登录

#### 2. 登录查看
1. 选择"学生登录"
2. 输入账号密码
3. 进入学生界面

## 🐛 故障排除

### 常见问题及解决方案

#### Q1: 无法启动后端服务器
**A**: 
1. 检查 MongoDB 服务是否运行
2. 检查端口 3000 是否被占用
3. 检查 node_modules 是否安装

#### Q2: 注册时提示"系统未录入学生信息"
**A**: 
1. 检查输入的姓名是否在学生表中
2. 运行 `node seed-students.js` 生成测试数据
3. 查看可用学生姓名列表

#### Q3: 登录后页面空白
**A**: 
1. 检查浏览器控制台错误
2. 检查后端服务器是否运行
3. 清空浏览器缓存

#### Q4: 无法连接到数据库
**A**: 
1. 检查 MongoDB 服务状态
2. 检查 .env 文件中的连接字符串
3. 检查防火墙设置
