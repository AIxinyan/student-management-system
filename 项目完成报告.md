**解决方案**：
1. 下载安装 Git for Windows
2. 配置环境变量
3. 重启 PowerShell

### 问题 3：PowerShell 语法差异 ❌

**现象**：
```powershell
mkdir frontend backend  # 报错
```

**原因**：PowerShell 的 mkdir 语法与 Linux 不同

**解决方案**：
```powershell
New-Item -ItemType Directory -Path frontend -Force
New-Item -ItemType Directory -Path backend -Force
```

**AI 辅助**：快速识别问题并提供正确语法
